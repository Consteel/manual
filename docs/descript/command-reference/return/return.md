<!-- wp:paragraph -->

Exits the script run.

<!-- /wp:paragraph -->

<!-- wp:heading {"level":3} -->

### Syntax

<!-- /wp:heading -->

<!-- wp:paragraph -->

**RETURN**

<!-- /wp:paragraph -->

<!-- wp:heading {"level":3} -->

### Description

<!-- /wp:heading -->

<!-- wp:paragraph -->

The RETURN command stops the script run, but leaves Consteel open. Code lines after the RETURN command are not going to be executed.

<!-- /wp:paragraph -->

<!-- wp:heading {"level":3} -->

### Sample code:

<!-- /wp:heading -->

<!-- wp:loos-hcb/code-block -->

```
RETURN
```

<!-- /wp:loos-hcb/code-block -->
