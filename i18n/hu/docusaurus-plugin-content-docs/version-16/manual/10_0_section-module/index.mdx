# Keresztmetszet modul  {/* Ez a sor az adott főcímért mint eddig */}

A Consteel számos funkcionalitása a keresztmetszetekhez kapcsolódik, ezért ezek egy különálló modulba, a Keresztmetszet modulba (csSection) kerültek. Ez a modul tartalmazza az összes olyan eszközt, ami keresztmetszet létrehozáshoz, keresztmetszeti tulajdonságok és feszültségek számításához, illetve a keresztmetszetek szabványos vizsgálatához tartoznak. Két párhuzamos keresztmetszet modellezés létezik egymás mellett: az Általános szelvénymodell (General Solid Section GSS) egy pontos, végeselem alapú felületmodell, bármilyen típusú keresztmetszethez; a Vékonyfalú szelvénymodell (Elastic Plate Segment, EPS) egy vékony lemezszegmensekből előállított szelvénymodell, speciálisan acél szelvényekhez. Az acél szabványos tervezési képletek az EPS modell alapján könnyen megoldhatók, beleértve a szerkezeti analízis során meghatározott keresztmetszeti igénybevételek alapján előállított tényleges 4. osztályú hatékony keresztmetszetet is.
